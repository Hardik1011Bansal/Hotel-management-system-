# Hotel Management System

A console-based hotel management system built using Python and MongoDB.

## Features
- Customer management (add, update, delete)
- Room management and availability tracking
- Booking system with check-in, check-out, and cancellation

## Technologies Used
- Python
- MongoDB (PyMongo)

## How to Run
1. Make sure MongoDB is running locally
2. Run main.py
